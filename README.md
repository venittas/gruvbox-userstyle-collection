# gruvbox-userstyle-collection
This is a collection of my userstyles for different websites with the gruvbox theme. This aims to make as many themes as possible for different websites. Requests won't be entertained but you are free to contribute if you want!

## Design
1. Gruvbox theme
3. Sharp corners (no roundess)
